Tipos de dados e variáveis

Conceitos:

1. Variável é um espaço delimitado dentro da memória do compudador.

Exemplo:

x = 1

Leia-se: x recebe 1

x = null

Leia-se: x recebe null (nulo)

*O Java Script recebe valores nulos

Dar uma valor para uma variável é chamado de atribuição.

O nome de uma variável é o seu identificador.

Regras:

1. Pode começar com letra, $ ou _;

2. Não pode começar com número;

3. É possível usar letras ou números;

4. É possível usar acentos ou símbolos;

5. Não pode conter espaços;

6. Não podem conter palavras reservadas.

Dicas para escolher nomes para identificadores:

1. Maiúsculas e minúsculas fazem diferença;

2. Tente escolher nomes coerentes para suas variáveis;

3. Evite se tornar um programador alfabeto ou um programador contador;

Em síntese:

VARIÁVEIS SERVEM PARA GUARDAR DADOS!!!

No JavaScript:

Existem duas palavras reservadas para variáveis no JS:

1. let

2. var

Tipos de dados:

1. Números inteiros (int)

2. Números reais (float)

Esses dois tipos de dados em JS se enquandram na classe de números. Ele não vai diferenciar entre um número inteiro e um número real.

3. Classe number:

1. Infinity

2. NaN

4. Strings, cadeias de caracteres

5. Booleano: True e False.

**O booleano é o mais primitivo de todos, só lembrar da questão do binário.

6. null

7. undefined

8. Object

1. Array;

9. Function

** Para saber qual é a natureza da sua variável utilize o comando: typeof "o nome da variável "
